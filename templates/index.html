Template Library

Reusable architecture templates

These templates are designed to be practical working assets. They can be used directly in markdown, adapted into SharePoint lists, wrapped in Power Apps workflows, or linked to relationship records in LeanIX.

How to use this library

Browse visually

Use the template pages when you want to understand purpose, structure, and where each asset fits.

Use the markdown

Each template also exists as a markdown file that can be copied into SharePoint, Git, Notion, or documentation tooling.

Operationalize later

These same structures can become intake forms, review workflows, and decision records in Power Apps or SharePoint Lists.
